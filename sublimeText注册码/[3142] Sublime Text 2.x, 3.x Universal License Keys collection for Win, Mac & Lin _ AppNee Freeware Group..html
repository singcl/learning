// Tips //

How to disable disable Sublime Text’s update check (new version prompt):

  1. Go to “Preferences -> Settings-User/Distraction Free“
  2. Add this line: “update_check”: false,
  3. Done!

If it does not work, add this line to hosts file:

  • 127.0.0.1 www.sublimetext.com

*** Windows: c:/windows/system32/drivers/etc/hosts

*** Linux: /etc/hosts
